Manual Data Tiles Request

Users who want to download data tiles must go through the data request process as described in Help - Data Request Registration Process, the user must e-mail the LiPAD Team <lipad@dream.upd.edu.ph> to notify the LiPAD Team of the intent to download the blocks within their uploaded shapefile. Available personnel will evaluate the data request and upload the requested data tiles to the FTP server.

After the data requested is uploaded to the FTP server, the personnel will e-mail the user notifying them that their data request is already available for downloading from the FTP server. The e-mail will contain instructions on how to access LiPAD's FTP server. Instructions to access the FTP server is also available at Help - Download files using FTP

Note: Some data requests will need approval from the Program Leader before the data request is uploaded to the FTP server.
